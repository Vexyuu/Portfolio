Projet 1

ShopWise

📌 description

ShopWise est une plateforme interactive qui permet aux utilisateurs de parcourir et filtrer des produits en fonction de leurs préférences. Le projet repose sur une interface dynamique qui affiche des marques et leurs produits associés.

🛠️ Technologies utilisées

  • HTML, CSS, JavaScript : Structure et style du site.
  • Fetch API : Pour récupérer dynamiquement les marques et produits depuis une base de données.
  • PHP & MySQL : Gestion des données avec WampServer64 et PHPMyAdmin.

🎯 Objectifs

  • Créer une expérience utilisateur fluide avec un affichage dynamique des produits.
  • Mettre en place un système de filtrage efficace.
  • Améliorer l'interaction avec des animations et une navigation intuitive.

Projet 2

Password Generator

📌 description

Password Generator est un outil permettant de générer des mots de passe sécurisés en fonction des préférences de l'utilisateur (longueur, présence de caractères spéciaux, chiffres, majuscules/minuscules).

🛠️ Technologies utilisées

  • HTML, CSS, JavaScript : Interface simple et intuitive.
  • JavaScript (Math.random) : Génération aléatoire des caractères.
  • Clipboard API : Permet de copier directement le mot de passe généré.

🎯 Objectifs

  • Offrir un moyen rapide et sécurisé de générer des mots de passe robustes.
  • Permettre à l’utilisateur de personnaliser ses préférences.
  • Assurer une interface ergonomique et responsive.
